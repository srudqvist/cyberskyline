# HTTP (Easy) 

## How I found the answers

Download the *HTTP2.pcap* file.  
run `source /etc/profile.d/zeek.sh` to reload the file path for zeek.  
run `zeek -C -r HTTP2.pcap`.  
run `head http.log` to see the *linux tool* and *IP* addresses.
